FastApi Calculadora Simple

FCS es una api con varias funcionalidades agregadas como cálculo básico, trigonometría, estadística, conversores, divisas, entre otros. Está implementada en Python con el framework FastApi.

  • Cálculo básico

    Las principales operaciones disponibles son sumar, restar, multiplicar, dividir, módulo, radicalización, potenciación y logaritmación

  • Conversión

    Las principales conversiones disponibles son ángulo, área, datos, longitud, masa, velocidad, sistema numérico, temperatura, tiempo, volumen y divisas

  • Trigonometría

    Las principales operaciones disponibles son seno, coseno, tangente, cotangente, secante y cosecante

  • Calculadoras

    Las principales operaciones disponibles son evaluación y cálculo de expresiones, cálculo de descuento y calculadora de IMC

  • Estadística

    Las principales operaciones disponibles son media, mediana, multi-moda, desviación estándar y varianza (población y muestra), correlación y covarianza
